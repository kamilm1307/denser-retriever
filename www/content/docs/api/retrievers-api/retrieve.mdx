## Retrieve

Send a query to the retriever to retrieve passages.

<APIPlayground method={"POST"} route={"/retrievers/retrieve"} bodyType={"json"} path={[]} query={[]} header={[{"name":"Authorization","value":"Bearer dr_-UcyDSXUzXvXt3fw-n8HB9Mf"}]} body={{"type":"object","isRequired":true,"description":"Send a query to the retriever to retrieve passages.","properties":{"query":{"type":"ref","isRequired":true,"schema":"0"},"k":{"type":"ref","isRequired":false,"schema":"1"},"id":{"type":"ref","isRequired":true,"schema":"2"}}}} schemas={{"0":{"type":"string","defaultValue":"Is denser free?","isRequired":false,"description":""},"1":{"type":"number","defaultValue":"4","isRequired":true,"description":""},"2":{"type":"string","defaultValue":"clzm3uwy000046t97v6iwyg8w", "isRequired":true,"description":""}}}>

</APIPlayground>

### Request Body

<Property name={"query"} type={"string"} required={true} deprecated={undefined}>

The query to send to the retriever.

<span>Example: `"Is denser free?"`</span>

</Property>

<Property name={"id"} type={"string"} required={true} deprecated={undefined}>

Identifier for a retriever.

<span>Example: `"clz957t700007xhx72s0aieda"`</span>

<span>Format: `"string"`</span>

</Property>

<Property name={"k"} type={"string"} required={true} deprecated={undefined}>

Retrieve top `k` passages.

<span>Example: `"4"`</span>

<span>Format: `"int"`</span>

</Property>

| Status code | Description |
| ----------- | ----------- |
| `200` | Success |
| `401` | Bad request |
| `500` | Internal server error |

</APIInfo>

```bash

  --header 'Authorization: Bearer dr_-UcyDSXUzXvXt3fw-nxxxxx' \
  --header 'content-type: application/json' \
  --data '{
  "query": "is denser free?",

  "k": 4
}'
```
